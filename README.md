# VK_Terminal

Web version of a Voight-Kampff terminal UI for use on Raspberry Pi with Chromium in kiosk mode.

## Project links
- Prop model parts (Google Drive): https://drive.google.com/drive/folders/1fALbdFYIXY8wNwbpdX9JjHC8JK31wbYh?usp=sharing

## Goals (draft)
- Run full-screen in Chromium kiosk on Raspberry Pi
- Simple deploy process (copy folder, open index.html or serve locally)

## How to run (local)
- Open `index.html` in a browser, or serve the folder with any static file server.
